
# 仿 10086 项目

### 安装依赖

```js
npm install // 安装前端项目依赖
cd server
npm install // 安装后端依赖
```

### 运行项目

```js
npm run serve // 运行前端项目
npm run dev // 运行后端项目
```

### 运用技术

```
vue
axios
通过自己封装组件实现页面布局
通过mockjs实现模拟前后端交互
```
